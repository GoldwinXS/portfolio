# goldwinxs.github.io/portfolio

Personal portfolio site built with React and Vite. Features a filterable project gallery, animated hero section, and dark/light mode based on system preference.

## Tech Stack

- React + Vite
- Material UI
- Framer Motion

## Features

- Dark/light mode (follows system preference)
- Filterable project cards by technology
- Animated hero with Typewriter effect
- Responsive layout

## Running Locally

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
```

Output goes to `dist/`. The site is deployed via GitHub Pages.
